# 💰 WalletOops

![Status](https://img.shields.io/badge/status-em%20desenvolvimento-yellow)
![License](https://img.shields.io/badge/license-MIT-blue)

**WalletOops** é uma aplicação web de gestão financeira pessoal. Regista despesas, organiza por categorias e visualiza os teus gastos através de gráficos intuitivos. Simples, rápido e direto ao ponto.
Premite sincronização com mais contas e formar uma grupo onde a informação é partilhada.

---

## ✨ Funcionalidades

- 📝 **Registo de despesas** — Adiciona gastos com valor, data e categoria
- 🏷️ **Categorização** — Organiza despesas por categorias personalizáveis
- 📊 **Gráficos e estatísticas** — Visualiza para onde vai o teu dinheiro
- 🌐 **Dados na nuvem** — Sincronização e backup com Supabase
- 📱 App para dispositivos móveis.

---

| Tecnologia | Descrição |
